It was built in the [React course](https://www.udemy.com/course/react-redux/) mentioned and features a custom hook that extracts video-fetching logic and makes it available to all components. A key adaptation I made was to create mobile and widescreen modes that are set via component state.

## Key skills

- React, React Custom Hooks
- Google YouTube API calls
