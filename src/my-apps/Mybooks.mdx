The app was adapted from a [React/Redux course](https://www.udemy.com/course/react-redux/). Users can authenticate via Google, search via the Google Books API and add links, and edit, update and delete books.

## Key skills

- React, React Hooks, ES5 and ES6 syntax
- Redux, Redux-Thunk
- Google Auth, Google Books API via Axios
- React Final Form
- Heroku CLI & hosting
